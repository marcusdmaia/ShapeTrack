# Directory Structure

## Root Level
- `index.html`: Entry point (Login/Landing).
- `dashboard.html`: Main mentor console.
- `vendas.html`: Sales management module.
- `estoque.html`: Inventory and borrowing tracking.
- `relatorio.html`: Performance and biometric reporting.
- `style.css`: Unified Design System and responsive rules.

## Directories
- `js/`:
  - `config.js`: App initialization, Supabase client, and menu logic.
- `supabase/`:
  - Contains database schema and migration notes.
- `.agent/`:
  - GSD system skills and agents.
- `.planning/`:
  - Codebase maps and project state.

## Key Modules
- **CRM**: `crm_dashboard.html`, `leads_admin.html`.
- **Biometrics**: `lancar_avaliacao.html`, `relatorio.html`.
- **Media**: `videos.html`, `parcerias.html`.
