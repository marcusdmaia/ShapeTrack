# Technology Stack

## Core Technologies
- **Frontend**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Custom Design System: Ethereal Glass)
- **Icons**: Phosphor Icons (via CDN)
- **Charts**: Chart.js (via CDN)

## Backend & Infrastructure
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Hosting**: Vercel (referenced in session logs)

## Key Dependencies (CDN)
- `https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2`
- `https://unpkg.com/@phosphor-icons/web`
- `https://cdn.jsdelivr.net/npm/chart.js`

## Configuration
- `js/config.js`: Centralized configuration for Supabase client and navigation menu.
- `style.css`: Main design system tokens and responsive layout rules.
